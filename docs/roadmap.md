# Roadmap inicial

## Fase 1: Fundamentos

- Definir stack técnico.
- Crear estructura del proyecto.
- Configurar base de datos.
- Crear autenticación básica.
- Crear roles iniciales.

## Fase 2: Control documental

- Crear módulo de documentos.
- Crear categorías.
- Subir archivos.
- Registrar versiones.
- Mostrar historial.

## Fase 3: Aprobaciones

- Crear bandeja de aprobaciones.
- Aprobar documentos.
- Rechazar documentos.
- Registrar comentarios.
- Generar tareas de aprobación.

## Fase 4: Acciones y tareas

- Crear módulo de acciones.
- Asignar responsables.
- Registrar seguimiento.
- Mostrar tareas pendientes.
- Marcar tareas como completadas.

## Fase 5: Dashboard

- Mostrar métricas principales.
- Mostrar actividad reciente.
- Mostrar tareas vencidas.
- Mostrar documentos pendientes.

## Fase 6: Preparación comercial

- Mejorar diseño visual.
- Crear datos demo.
- Preparar una demo guiada.
- Definir planes y precios iniciales.
